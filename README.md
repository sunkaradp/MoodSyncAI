# MoodSyncAI

MoodSyncAI is a Multi-Modal Sentiment and Emotion Analyzer built using Deep Learning and Generative AI.

## Features

- Facial emotion recognition using CNN
- Text sentiment analysis using Transformer
- Multimodal fusion
- Emotion mismatch detection
- AI-generated emotional summaries
- Streamlit user interface

## Technologies Used

- Python
- Streamlit
- DeepFace
- Hugging Face Transformers
- TensorFlow
- PyTorch

## Models Used

| Task | Model |
|------|------|
| Facial Emotion Recognition | DeepFace CNN |
| Text Sentiment Analysis | DistilBERT |
| Generative AI Summary | FLAN-T5 |
